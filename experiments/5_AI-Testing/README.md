# Experiment 5: AI Browser Testing

## Useful Tools

- [Comet](https://www.perplexity.ai/comet) — AI Browser by Perplexity
- [Atlas](https://chatgpt.com/atlas) — AI Browser by OpenAI
- [Claude for Chrome](https://claude.com/claude-for-chrome) — Claude AI browser extension
- [Gemini Sidebar](https://chromewebstore.google.com/detail/sidebar-with-gemini/ebamemkjoacjajcmhjjkcnngpafbkdbh) — Google Gemini built into Chrome sidebar

## Try It Yourself

1. Install one of the AI browsers listed above (Comet or Atlas recommended)
2. Use it to explore [Foodora clone app](https://foodora.lovable.app) — try to find a bug using the AI assistant
3. Try the same task with a second AI tool (e.g., Claude for Chrome) — compare the findings
